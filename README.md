# 📊 Telco Customer Churn Analysis Platform

A dynamic full-stack web application designed to track, visualize, and analyze telecom customer demographics, service usage, and churning patterns. This platform helps businesses identify high-risk accounts and leverage data-driven insights to maximize retention strategies.

---

## 🚀 Live Demo
Check out the deployed application here: **[https://customer-churn-analysis-vheh.onrender.com]**

---

## ✨ Features
* **Interactive Dashboard:** Real-time visualization of key metric performance indicators (KPIs) like overall churn rate, average monthly charges, and total revenue loss.
* **Funnel & Conversion Tracking:** Deep dive into customer lifecycle metrics from initial onboarding down to account cancellation triggers.
* **Dynamic Filtering:** Toggle view settings across customer contract categories (Month-to-month, One year, Two year) and active payment streams.
* **Responsive Frontend:** Beautiful, fully interactive data dashboard tailored for modern desktop and mobile viewports.

---

## 🛠️ Tech Stack

### Frontend
* **React** with **TypeScript** (For robust, type-safe interactive components)
* **Vite** (Ultra-fast modern frontend tooling)
* **Tailwind CSS** / UI Frameworks (For clean, scannable, and modern styling Layouts)

### Backend & Deployment
* **Node.js** with **TypeScript** (`server.ts`)
* **Express** (Robust backend API routing framework)
* **Render Cloud Platform** (For continuous integration and cloud web hosting)
